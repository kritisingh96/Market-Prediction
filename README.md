# Hull Tactical Market Prediction

This repository contains a Kaggle notebook for the Hull Tactical Market Prediction competition.

## Notebook

- [Market Prediction Notebook](https://www.kaggle.com/code/kriti83/market-prediction?scriptVersionId=279974718)

## Overview

- **Goal:** Predict tactical market allocations.
- **Data:** Uses train/test datasets provided by the competition.
- **Model:** Random Forest Regressor.
- **Features:** Lagged returns, rolling statistics, momentum, volatility ratio.
